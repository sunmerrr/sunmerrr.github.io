## 갑자기 분뇨가 Web3
### 풀어야하는 미션
그냥 web3를 개발한다기보다는 그냥 일반 웹(web2)처럼 보이지만 그 안에 자연스럽게 web3가 녹아져 있어야 했다.     
첫번째 단계로서 유저가 코인을 상금으로 걸고, 타 유저들이 해당 이벤트에 참여해서 상금을 받아 갈 수 있도록 해야했다.     
     
### 어디까지 해봤니
1. 강의 듣기     
코인쪽은 1도 모르는 나라서 그냥 일단 udemy 강의를 샀다. 사서 무작정 따라햇다.      
solidity 가 뭔지도 모르고, Remix 의 존재 유무도 모른채 그냥 강의 따라서 코드를 쳤다. 그렇게 강의만 따라가니 내가 작성한 스마트 콘트렉트로 테스트 거래가 가능했다.(강의 만세)     

1. 경력 개발자 괴롭히기      
회사에 마침 관련해서 개발하던 천재 개발자가 있었다. 그분에게 거의 같은말을 백번쯤 들을 때까지 똑같은것을 물어봤던것 같다. 용어도 생소하고 신경쓸것도 많아서 한번에 모든 흐름이 머리속에 들어오지 않았다.(다행히 101번까지도 같은 발을 반복해서 해주셨다. 감사해요 모글리(닮음)) 

1. 삽질하기
- ABI 받았냐는 질문에 API 는 이미 있었다고 답하기
- 주소를 `0x0000000000000000000000000000000000000000` 로 넣어서 배포하기
- 가스비를 개 높게 잡아서 테스트 할때 사용해야하는 코인 다 쓰기(2번의 그분에게 구걸해서 더 받음)
- 환경변수 세팅 안하고 안된다고 2번의 그분 찾아가기
- 냅다 BigInt 사용해서 에러내기     
(뭐 삽질로 에러 낸 게 너무나 많아서 다 적을 수는 없을 것 같다.ㅠ)
